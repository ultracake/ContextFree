The folder antlr contains:
	1. The source code to Mandatory Assignment 2, and the Calculator example
	2. antlr v. 3.5
	3. antlr.bat which can be used for running antlr+java in windows cmd
Make sure that jdk is installed, and an environment variable is set for javac and java.
antlr.bat works is executed in the following way:
	1. open cmd in the antlr folder: navigate to antlr folder in file explorer press Ctrl+Shift+RIGHTCLICK and press open command window here
	2. the program can be executed by: antlr [While/Calc] [TEST]. For example antlr While Program1.while, antlr Calculator example.calc.
	   The TEST files is found in antlr/test.
